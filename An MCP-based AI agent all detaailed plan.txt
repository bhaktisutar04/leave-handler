An MCP-based AI agent powered by Groq (Llama 3.3-70b). Groq is the brain — it reads leave emails, reasons about conflicts and policy, and decides what to do. Gmail, Calendar, and Google Chat are its MCP tools — the hands it uses to take real actions. Python is just the runtime connecting everything. Deployed free on GitHub Actions.

agent.py — the MCP loop (the heart of the project)
This is where MCP happens. Groq decides which tool to call. Python executes it. Result goes back to Groq. Repeats until Groq has no more tool calls.
import json
from groq import Groq
from config import GROQ_API_KEY
from tool_schemas import MCP_TOOLS
from prompts import SYSTEM_PROMPT
from tools import execute_tool

def run_agent():
    client = Groq(api_key=GROQ_API_KEY)
    messages = [
        {"role": "system",  "content": SYSTEM_PROMPT},
        {"role": "user",    "content": "Process all leave requests now."}
    ]

    print("Agent starting...\n")

    while True:
        response = client.chat.completions.create(
            model="llama-3.3-70b-versatile",
            messages=messages,
            tools=MCP_TOOLS,
            tool_choice="auto"
        )
        msg = response.choices[0].message

        if msg.tool_calls:
            messages.append(msg)
            for tc in msg.tool_calls:
                name   = tc.function.name
                args   = json.loads(tc.function.arguments)
                print(f"Groq calling tool: {name}({args})")
                result = execute_tool(name, args)
                print(f"Tool result: {result}\n")
                messages.append({
                    "role":        "tool",
                    "tool_call_id": tc.id,
                    "content":     json.dumps(result)
                })
        else:
            print("Agent finished.\n")
            print(msg.content)
            break

GitHub Actions workflow — run.yml
Create this file at .github/workflows/run.yml in your repo. It runs main.py every Monday at 8am IST (2:30am UTC) automatically, for free, forever.
name: Leave Request Handler

on:
  schedule:
    - cron: "30 2 * * MON"   # 8:00am IST every Monday
  workflow_dispatch:          # Also allows manual trigger

jobs:
  run:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - uses: actions/setup-python@v4
        with:
          python-version: "3.11"

      - name: Install dependencies
        run: pip install -r requirements.txt

      - name: Write credentials files
        run: |
          echo '${{ secrets.GOOGLE_CREDENTIALS }}' > credentials.json
          echo '${{ secrets.GOOGLE_TOKEN }}'       > token.json

      - name: Run agent
        env:
          GROQ_API_KEY:      ${{ secrets.GROQ_API_KEY }}
          CHAT_WEBHOOK_URL:  ${{ secrets.CHAT_WEBHOOK_URL }}
        run: python main.py
